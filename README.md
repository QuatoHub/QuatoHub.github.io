## Getting Started

#### Installation

1. Clone the Repo

```shell
git clone https://github.com/mnxmnz/mnxmnz.github.io.git
```

```shell
cd mnxmnz.github.io
```

2. Install Project Packages

```shell
npm install
```

3. Run the Project (Dev Mode)

```shell
gatsby develop

http://localhost:8000/
```
